路透社資料圖片

港府今年9月起分階段啟用「預先通報乘客資料系統」，預料到明年入境香港或經香港機場過境的航機乘客，必須先獲香港入境處的「可登機指示」才能上機。保安局長鄧炳強周二（5日）在立法會表示，預報系統可事前阻截「免遣返聲請」人士來港，又聲稱是國際民航規定。本台翻查《國際民航公約》，發現鄧炳強說法並非完全真確；亦有前社工質疑，有關預報系統只是為阻止政治上「不受歡迎」的人士來港，因大部分聲請者都是經水路非法入境，預報系統對其根本沒影響。

香港立法會保安事務委員會周二（5日）進行《施政報告》政策簡報會，其中保安局局長鄧炳強交代處理「免遣返聲請」人士來港的措施，當中包括加強與大陸及澳門的情報交換、進一步加強打擊黑工，以及一個正分階段實行的新措施。

鄧炳強說：「我們亦透過2024年9月份起，分階段實施的『預先通報乘客資料系統』，加強阻截可能濫用免遣返聲請機制的人，去乘搭航班來港。」

鄧炳強：國際民航規定設預報系統　指引：互動式預報可有可無

但鄧炳強其後回應委員會副主席簡慧敏時，疑似「鬼拍後尾枕」，道出系統的真正用途：「 而我們這個預先申報系統，首先第一就是國際民航的規定；第二就是由於他（旅客）來之前，我們已經知道他的身分是誰。如果有些原因我們認為他不應該被入境的話，也預先透過這個系統，可以告訴（在）出發地的旅客，從而（令航空公司）考慮是否將人載來香港，因為來香港之後可能（將旅客）載回出發地。」

國際民航組織2018年為應對國際恐怖主義及極端主義修訂《國際民航公約》，規定締約國必須建立「預報乘客資料系統」（Batch API system）；但港府採用的是可發出登機許可（Board directives）的「互動式預報系統」（Interactive API system），而互動式預報並非國際民航組織的硬性規定。另外，鄧炳強聲稱有關系統可讓航空公司及旅客「考慮」是否來港，但根據入境處有關資料，除非入境處發出「可登機指示」，否則航空公司根本不得讓旅客上機。

前社工：聲請者多經水路非法入境　質疑港府系統「政治先行」

前香港社工王惠芬在港時非常關注聲請者及難民議題，她質疑有關系統能否如鄧炳強所說，加強堵截「免遣返」聲請者。

王惠芬說：「其實絕大部分的來到香港呈請難民庇護的，基本上是經水路，就是在中國內地，尤其是廣州那些，過來香港一些不同的地方去上岸的，例如西貢。所以其實現在（港府）新的拒絕一些不受歡迎人物這個機制，其實是不能夠真的堵截到他們的。」

王惠芬認為，該預報系統其實是為了防止因為「其他原因」，而成為「不受港府歡迎的人」登機抵港，她說：「我就覺得這個機制是用來防止一些在政治方面，支持香港的民主、人權、自由，那些特區政府不歡迎的人到達香港邊境……你扣留他們，跟住盤問他到某個時間，接著要他原機遣返。這個是比較難看的，也都會受到一個國際社會的譴責或者嘲笑。」

葛珮帆：隨時有聲請者在街頭行劫　王惠芬：一竹竿打一船人不公道

另外，立法會議員葛珮帆在委員會上亦提到，憂慮「免遣返聲請」留港者會影響香港治安：「最近有一位警員在街上被人用刀襲擊，刺了5刀，而那位行兇者是持『行街紙』的、申請了免遣返聲請的人士，令很多市民更加嘩然，原來隨時會有這些人持刀在街頭行劫。」

王惠芬承認，無可否認是有「免遣返聲請」人士可能為生活而選擇犯罪，但認為建制派一直以「一竹竿打一船人」的說法並不公道。

她說：「任何一單事情發生，他們就說那些『假難民』影響治安很嚴重。我覺得他們這樣說因為很方便，因為對於一些沒有身分證的人士、或者在尋求一些庇護的人，他們是無聲的一群、是最弱勢的一群，你怎麼抹黑他們、怎麼說他們，其實都沒有人會代他們出聲，這是最容易的。」

鄧炳強指，目前只有5%航空公司正在使用「預先通報乘客資料系統」，預料於年底會多一半航空公司加入，明年第二季就會覆蓋所有在香港國際機場營運的航空公司。
